RELAX NG Non-XML Syntax

I have developed an experimental non-XML syntax for RELAX NG. The following are available:

  • a description of the syntax
  • an example showing a schema for RELAX NG's XML syntax written in the non-XML syntax (this is optimized for readability in the non-XML syntax and is not simply a translation of relaxng.rng)

The following tools provide support for the syntax:

  • Jing will validate using the non-XML syntax when the -n option is used
  • Trang can translate schemas written in the non-XML syntax; in particular, trang can be used to translate schemas written in the non-XML syntax into the standard RELAX NG XML syntax

This syntax is not a part of RELAX NG, and is not a product of the OASIS RELAX NG TC. However, the OASIS RELAX NG TC have decided to produce a non-XML syntax for RELAX NG, which I expect to be similar to this syntax.
